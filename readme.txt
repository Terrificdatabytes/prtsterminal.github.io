Installation: move the com.rockstargamesgtasa folder along the Android/data path with the replacement.
Rename AML.apkk to AML.apk and install it.

Recommendations:
1. You need an obb cache to run the mod!
2. Set the brightness in the game settings below 20%.
3. If the game crashes during the download phase, delete the CINFO.BIN file, which is created along the path Android/data/com.rockstargamesgtasa/files, and MINFO.BIN, which is created along the path Android/data/com.rockstargamesgtasa/files/models. 
4. If crashes are observed in the game, try deleting the CINFO.BIN file, which is created along the path Android/data/com.rockstargamesgtasa/files, and MINFO.BIN, which is created along the path Android/data/com.rockstargamesgtasa/files/models.
5. If the game on your device crashes, try going to the path "configs ->net.rusjj.jpatch.ini -> RLEDecompressCrashFix" and change the value to 1 to try to fix the crash.
6. If you change the value to 1 on the path "configs -> net.rusjj.jpatch.ini -> BackPolesLightShadow", you can turn on the lights from the lanterns as on PC/PS2, but this may cause game crashes.
7. You can correct the viewing angle in cutscenes by going to the path "configs ->net.rusjj.jpatch.ini ->FixCutsceneFOV" and changing the value to 1, but this may not work correctly on your device.

You can follow the changes in the mod here: https://t.me/+tWvQeSK3kwQwNmEy


Установка: перекинуть папку com.rockstargamesgtasa по пути Android/data с заменой.
Переименуйте AML.apkk в AML.apk и установите его. 


Советы:
1. Для запуска мода вам нужен obb кэш! 
2. Поставить яркость в настройках игры ниже 20%.
3. Если игра вылетает на этапе загрузки, удалите файл CINFO.BIN, который создаётся по пути Android/data/com.rockstargamesgtasa/files, и MINFO.BIN, который создаётся по пути Android/data/com.rockstargamesgtasa/files/models. 
4. Если в игре наблюдаются вылеты, попробуйте удалить файл CINFO.BIN, который создаётся по пути Android/data/com.rockstargamesgtasa/files, и MINFO.BIN, который создаётся по пути Android/data/com.rockstargamesgtasa/files/models.
5. Если сборка на вашем устройстве вылетает, попробуйте перейти по пути "configs -> net.rusjj.jpatch.ini -> RLEDecompressCrashFix" и поменять значение на 1, чтобы попробовать исправить вылет.
6. Если по пути "configs -> net.rusjj.jpatch.ini -> BackPolesLightShadow" поменять значение на 1, вы сможете включить свет от фонарей как на PC/PS2, но это может вызвать вылеты игры.
7. Вы можете исправить угол обзора в катсценах, перейдя по пути "configs -> net.rusjj.jpatch.ini -> FixCutsceneFOV" и изменив значение на 1, но это может некорректно работать на вашем устройстве.
